## 1. ゲーム概要

### コンセプト

2人のプレイヤーが**波を生成・発射**し、フィールド上で発生する**合成波の干渉現象**を利用して相手にダメージを与える対戦ゲーム。

物理の「波の重ね合わせ」をゲームメカニクスに昇華させたオリジナル体験を提供する。

### 基本ルール

- 各プレイヤーは波のパラメータ（種類・振幅・周波数・位相）を調整して発射する
- 2つの波がフィールド中央で衝突すると合成波が発生する
- 合成波の振幅がダメージ値を決定する
- HPが0になったプレイヤーの負け

### 対象プラットフォーム

| プラットフォーム | 操作方式 |
|---|---|
| PC (Windows / macOS / Linux) | キーボード + マウス |
| モバイル (iOS / Android) | タッチ + フリック |

## 2. コアメカニクス

### 波の重ね合わせ

フィールド上の合成波は以下の式で計算する。

```
y(x, t) = A1·f1(k1·x - ω1·t + φ1) + A2·f2(k2·x - ω2·t + φ2)
```

| 変数 | 意味 |
|---|---|
| A | 振幅 (0.1 〜 3.0) |
| f | 波の種類関数 (sin / cos / tan など) |
| k | 波数 = 2π / λ |
| ω | 角周波数 = 2π·freq |
| φ | 位相 (0 〜 2π) |

### 干渉パターンと効果

| 干渉の状態 | 条件 | 効果 |
|---|---|---|
| **強め合い（建設的干渉）** | 同位相 | 大ダメージ |
| **弱め合い（破壊的干渉）** | 逆位相 (φ差 = π) | 相殺・防御 |
| **うなり** | 周波数が近似 | 連続・範囲ダメージ |
| **定常波** | 同周波数・逆向き | バリア生成 |

### ダメージ計算

```
damage = maxAmplitude(compositeWave) × damageCoefficient × waveTypeMultiplier
```

- `damageCoefficient` = 10（基本値）
- `waveTypeMultiplier` は波の種類相性で変動（後述）

## 3. 波の種類

### 一覧

| ID | 名称 | 数式 | 特性 | 戦略的役割 |
|---|---|---|---|---|
| `sin` | Sin波 | `sin(x)` | 滑らか・連続 | バランス型 |
| `cos` | Cos波 | `cos(x)` | 位相90°ずれ | カウンター型 |
| `tan` | Tan波 | `tan(x)` | 鋭い・不連続 | 高リスク貫通型 |
| `square` | 矩形波 | `sign(sin(x))` | 壁状・急峻 | 防御・バリア型 |
| `sawtooth` | 鋸歯波 | `(x mod 2π)/π - 1` | 一方向に鋭い | 連続攻撃型 |
| `triangle` | 三角波 | `2/π·arcsin(sin(x))` | 山が連続 | 安定・持続型 |

### 波種の相性（三すくみ）

```
Sin  →（有利）→  Tan
Tan  →（有利）→  矩形
矩形 →（有利）→  Sin

※ Cos・鋸歯・三角は中立（相性ボーナスなし）
```

相性有利の場合、`waveTypeMultiplier = 1.5`  
相性不利の場合、`waveTypeMultiplier = 0.7`

## 4. 操作仕様

### PC（キーボード）

| 操作 | Player 1 | Player 2 |
|---|---|---|
| 移動（左右） | `A` / `D` | `←` / `→` |
| 振幅調整（上下） | `W` / `S` | `↑` / `↓` |
| 周波数調整 | `Q` / `E` | `,` / `.` |
| 位相調整 | `Z` / `X` | `N` / `M` |
| 波の種類切替 | `1`〜`6` | `Num1`〜`Num6` |
| 波を発射 | `Space` | `Enter` |

### モバイル（タッチ・フリック）

| ジェスチャー | 効果 |
|---|---|
| **上方向フリック**（速度比例） | 振幅を増加 |
| **下方向フリック** | 振幅を減少 |
| **左右フリック**（速度比例） | 周波数を変更 |
| **回転スワイプ（時計回り）** | 位相を+方向にシフト |
| **回転スワイプ（反時計回り）** | 位相を−方向にシフト |
| **ピンチアウト** | 波長を伸ばす |
| **ピンチイン** | 波長を縮める |
| **波種アイコンタップ** | 波の種類を切替 |
| **発射ボタンタップ** | 波を発射 |

### パラメータ範囲

| パラメータ | 最小値 | 最大値 | デフォルト | 変化ステップ |
|---|---|---|---|---|
| 振幅 (A) | 0.1 | 3.0 | 1.0 | 0.1 |
| 周波数 (freq) | 0.5 | 5.0 | 1.0 | 0.1 |
| 位相 (φ) | 0 | 2π | 0 | π/8 |

## 5. 戦闘システム

### HP

- 初期HP：両プレイヤーともに **100**
- HPが0以下になったプレイヤーの負け

### 共鳴ゲージ

同じ波種を連続使用すると共鳴ゲージが蓄積する。

| 連続使用回数 | 共鳴レベル | 効果 |
|---|---|---|
| 0〜1回 | Lv0 | なし |
| 2〜3回 | Lv1 (★☆☆) | ダメージ +20% |
| 4〜5回 | Lv2 (★★☆) | ダメージ +40% |
| 6回以上 | Lv3 (★★★) | **必殺波**発動可能 |

必殺波発動後、ゲージはリセットされる。

### 合成波プレビュー

波を発射する前に、**リアルタイムで合成波の波形を画面下部に表示**する。プレイヤーはダメージ量を事前に予測できる。

### 環境波（フィールドギミック）

フィールド背景に定期的に変化する環境波が存在し、自分の波との合成でダメージが増減する。

- 環境波周期：10秒ごとに変化
- 影響：ダメージ ±30%

## 8. 技術仕様

### 波の描画

- 1フレームごとに各波形をサンプリングして折れ線で描画
- サンプル数：画面幅に対して2px間隔（例：1280px → 640点）
- Tan波は不連続点付近（|tan(x)| > 10）でクリッピング

### 衝突・干渉計算

- 波がフィールド中央（x = 640）に到達したタイミングで合成計算を実行
- 合成波の最大振幅を10点サンプリングで近似

```go
func compositeAmplitude(w1, w2 Wave, t float64) float64 {
    max := 0.0
    for i := 0; i < 10; i++ {
        x := float64(i) * 0.1
        v := math.Abs(w1.ValueAt(x, t) + w2.ValueAt(x, t))
        if v > max { max = v }
    }
    return max
}
```

### フレームレート

- 目標：60 FPS
- 波アニメーション更新：毎フレーム
- 干渉計算：波発射時のみ（イベント駆動）
